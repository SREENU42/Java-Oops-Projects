# Student Grade System

## Description
This project demonstrates **Encapsulation** in Java and manages student marks for three subjects.  
It calculates total marks, average marks, and allows validation for input marks.

---

## Features
- Create student records with name and marks for three subjects.
- **Encapsulation**: Private fields with getters and setters.
- Validate marks (0â€“100 only).
- Calculate total marks.
- Calculate average marks.
- Display student details.

## Validation

- Invalid index or mark will display an error message.
