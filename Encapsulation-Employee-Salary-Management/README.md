# Employee Salary Management System

## Description
This project demonstrates **Encapsulation** in Java and simulates a simple employee salary management system. 
Employee salary is kept private and can only be accessed or modified through getter and setter methods. 
It also supports applying bonuses and deducting taxes with proper validation.

---

## Features
- Create employee records with ID, name, and salary.
- **Encapsulation**: Salary field is private and validated.
- Validate salary (cannot be below minimum wage of 15000).
- Apply **bonus** by percentage.
- Apply **tax** by percentage.
- Display employee details.

---

## Validation
- Salary cannot be set below **minimum wage (15000)**.
- Bonus and tax percentages must be positive.
